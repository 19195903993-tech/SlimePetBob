# 电子宠物史莱姆

本项目是一个纯前端独立版本小游戏，直接打开 `index.html` 即可运行。

## 初始化后的开发环境

- 本地静态服务器：`scripts/server.js`
- 启动命令：`npm run dev`
- 默认地址：[http://localhost:5173](http://localhost:5173)

## 快速开始

1. 安装 Node.js（建议 18+）
2. 在项目目录执行：

```bash
npm run dev
```

3. 浏览器访问 `http://localhost:5173`

## 目录结构

```text
.
├─ index.html
├─ package.json
```
